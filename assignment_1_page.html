Intranet

The intranet is a lot like the internet. The internet is a global network network of billions of computers and other electronic devices. Without the internet, you wouldn't be able to access or view any information. Just like the Internet, the intranet is a network of computers and other electronic devices. It is used by private businesses, schools or company. It can be website and is inaccessible to the public through the World Wide Web. The internet is publically accessible while the intranet isn't. The intranet alows you to engage with your company/school and collaborate with your colleagues, anytime, anywhere. Anyone that would want to access this network would need to have a special network password and be connected to LAN(Local Area Network). A firewall software is essential to protect private documents from the users. Firewall stands between the outside internet and the private intranet.

Internet Protocol

The Internet Protocol also known as IP is a set of rules which routes addressing packets of data so they can travel across networks and arrive at the same destination. When in the internet, it is divided into smaller pieces. These are called packets and are the reason why you have internet. An IP is used to helps routers send these packets to the right place. Everything that connects to the internet will have an IP address. All packets would be directed to these IP address on where internet and data is needed.

When searching up a website domain, you would almost always see HTTP at the start. In fact, HTTP is the backbone of the World Wide Web. It is an underlying protocol used by the WORLD WIDE WEB that defines how messages are formatted and transmitted. It also tells Web servers and browsers what should do throught various commands. When you enter an HTTP command to the web browser such as Chrome, it sends an HTTP command to the web server directing it to fetch and transmit the requested Web page.

An IP or internet protocol does not handle packet ordering or error checking. Instead, we use TCP or Transmission Control Panel which does that. TCP collects all the info given by the data connected with IP and it sorts its out into the right order. It also tells if there are missing pieces and tells that it has been received. Hence the relationship between them.
